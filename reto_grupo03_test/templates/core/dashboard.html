Example API Usage

Check Promotions

POST /api/check-promotions/

{
    "cliente_id": 1,
    "items": [
        {
            "articulo_id": "AC0353",
            "cantidad": 48,
            "precio_unitario": 100.00
        }
    ]
}

This endpoint checks applicable promotions for an order.

List Active Promotions

GET /api/promociones/

Lists all active promotions with their conditions and bonifications.

Curl Examples

# Check promotions for an order
curl -X POST http://localhost:8000/api/check-promotions/ \
    -H "Content-Type: application/json" \
    -d '{
        "cliente_id": 1,
        "items": [
            {
                "articulo_id": "AC0353",
                "cantidad": 48,
                "precio_unitario": 100.00
            }
        ]
    }'

# List all promotions
